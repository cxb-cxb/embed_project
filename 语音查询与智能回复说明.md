# 客户语音查询与智能回复演示说明

## 当前版本

本版本先实现“客户问题文本 -> 智能回复”的核心链路。麦克风 ASR 和扬声器 TTS 后续接入时，只需要把语音识别结果传给同一个脚本，再把回复文本送去播报即可。

## 双模式

- 默认 `auto`：优先调用联网大模型；没有 API Key、网络失败或接口异常时，自动切回离线关键词 + 模板回复。
- 稳定 `offline`：完全离线，使用关键词和商品目录生成回复，适合比赛现场保底演示。
- 强制 `llm`：只调用大模型，失败会直接报错，适合调接口。

## 一键启动

在 PowerShell 中进入项目目录：

```powershell
cd "C:\Users\33591\Documents\嵌赛 2"
```

启动默认双模式：

```powershell
.\start_voice_assistant_demo.ps1
```

只用离线保底模式：

```powershell
.\start_voice_assistant_demo.ps1 -Mode offline
```

指定当前视觉识别商品，例如当前画面是牛奶：

```powershell
.\start_voice_assistant_demo.ps1 -Mode offline -CurrentProduct milk
```

单句测试：

```powershell
.\start_voice_assistant_demo.ps1 -Mode offline -Question "牛奶多少钱"
```

## 联网大模型配置

默认使用 OpenAI 兼容接口。DeepSeek 可作为智能回复大模型使用：

```powershell
.\configure_deepseek_api.ps1
```

按提示输入 DeepSeek API Key 后，会写入当前 Windows 用户环境变量：

```text
RETAIL_LLM_API_KEY
RETAIL_LLM_BASE_URL=https://api.deepseek.com
RETAIL_LLM_MODEL=deepseek-v4-flash
```

重新打开 PowerShell 后测试：

```powershell
.\start_voice_assistant_demo.ps1 -Mode auto -Question "牛奶适合早餐吗"
```

也可以临时在当前 PowerShell 设置环境变量：

```powershell
$env:RETAIL_LLM_API_KEY="你的DeepSeek_API_KEY"
$env:RETAIL_LLM_BASE_URL="https://api.deepseek.com"
$env:RETAIL_LLM_MODEL="deepseek-v4-flash"
.\start_voice_assistant_demo.ps1 -Mode auto
```

如果使用其他 OpenAI 兼容平台，只需要替换 `RETAIL_LLM_BASE_URL` 和 `RETAIL_LLM_MODEL`。

注意：DeepSeek 负责“根据问题生成智能回复”，不负责 ASR 语音转文字。麦克风语音识别仍需要 OpenAI Whisper、FunASR、讯飞、火山引擎等 ASR 服务或离线模型。

## 后续接入麦克风和播报

后续升级路径：

1. 麦克风录音。
2. ASR 输出文字，例如“牛奶多少钱”。
3. 调用 `tools\voice_retail_assistant.py --question "牛奶多少钱"`。
4. 终端/LVDS 显示回复。
5. 接入扬声器后，把同一段回复文本送给 TTS 播放。

## 麦克风输入版

当前已增加“板子麦克风录音 -> 电脑端 ASR -> 终端回答”脚本：

```powershell
.\start_voice_mic_query_demo.ps1 -Seconds 5 -ReplyMode offline
```

运行后对着板载麦克风说话，例如“牛奶多少钱”。脚本会从板子录音并拉回电脑，再调用 ASR 转文字，然后把文字送入当前问答逻辑。

如果还没有配置 ASR Key，可以先用模拟 ASR 文本验证后半段：

```powershell
.\start_voice_mic_query_demo.ps1 -ReplyMode offline -AsrText "牛奶多少钱"
```

配置火山引擎豆包语音 ASR：

```powershell
.\configure_volcengine_asr.ps1
```

按提示输入火山引擎豆包语音 ASR 的 API Key / Access Token。配置后重新打开 PowerShell，测试：

```powershell
.\test_volcengine_asr.ps1 -Seconds 5
```

完整麦克风问答：

```powershell
.\start_voice_mic_query_demo.ps1 -Seconds 5 -AsrProvider volcengine -ReplyMode offline
```

如果已经配置 DeepSeek，则可以让 ASR 使用火山、回复使用 DeepSeek：

```powershell
.\start_voice_mic_query_demo.ps1 -Seconds 5 -AsrProvider volcengine -ReplyMode auto
```

## 视觉识别联动

视觉 LVDS 脚本会把当前识别结果写入：

```text
D:\qsm_embed_dataset\lvds_overlay_runtime\current_product.json
```

因此先启动视觉识别：

```powershell
.\start_visual_lvds_demo.ps1
```

再启动语音查询：

```powershell
.\start_voice_mic_query_loop.ps1 -Seconds 6 -ReplyMode auto
```

此时可以直接问：

```text
这个多少钱
这个还有库存吗
推荐一下这个商品
```

语音脚本会自动读取最近一次视觉识别商品作为 `CurrentProduct`。如果想忽略视觉识别结果，可以使用 `tools\voice_query_from_board_mic.py` 的 `--ignore-visual-state` 参数。
