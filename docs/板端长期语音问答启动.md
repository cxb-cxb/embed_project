# 板端长期语音问答启动

## 1. 一键启动

```bash
cd /userdata/Embed_project
sh scripts/run_voiceask.sh
```

进入程序后：

```text
voiceask 3
```

## 2. Wi-Fi 重连

如果重启后网络没通：

```bash
cd /userdata/Embed_project
sh scripts/connect_wifi.sh
```

## 3. 本地配置文件

板子上使用两个本地配置文件：

```text
/userdata/Embed_project/config/wifi.env
/userdata/Embed_project/config/voice_asr.env
```

这些文件保存热点密码和云端密钥，不提交到 GitHub。
