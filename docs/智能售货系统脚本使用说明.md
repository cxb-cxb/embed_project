# 智能售货系统脚本使用说明

本文档说明板端智能售货系统常用脚本的用途、启动方式、语音购物流程、付款码流程和排障方法。

## 1. 主要脚本

项目在板端默认路径：

```sh
/userdata/Embed_project
```

常用脚本：

```sh
/userdata/Embed_project/scripts/run_mission.sh
/userdata/Embed_project/scripts/start_retail_lvds_ui.sh
/userdata/Embed_project/scripts/start_voice_auto_listen.sh
/userdata/Embed_project/scripts/run_voiceask_speaker.sh
/userdata/Embed_project/scripts/restore_lvds_desktop.sh
```

脚本作用：

| 脚本 | 作用 |
| --- | --- |
| `run_mission.sh` | 一键启动完整演示系统：LVDS 售货界面 + 语音监听 |
| `start_retail_lvds_ui.sh` | 启动扫码/购物车/付款码界面，并带 watchdog 自动拉起 |
| `start_voice_auto_listen.sh` | 启动唤醒词监听流程 |
| `run_voiceask_speaker.sh` | 单次语音识别、语音购物车命令、语音播报 |
| `restore_lvds_desktop.sh` | 停止 LVDS 售货界面并恢复桌面 |

## 2. 一键启动

在电脑上通过 adb 启动：

```powershell
C:\tmp\adb.exe shell "sh /userdata/Embed_project/scripts/run_mission.sh"
```

在板子 shell 中启动：

```sh
sh /userdata/Embed_project/scripts/run_mission.sh
```

启动后会自动执行：

1. 停止旧的语音进程和播放进程。
2. 配置扬声器和麦克风。
3. 启动 LVDS 智能售货界面。
4. 启动语音唤醒监听。

正常输出会包含：

```text
[mission] run_mission is ready.
[mission] screen_log=/userdata/Embed_project/qr_scanner_display.log
[mission] voice_log=/tmp/qsm_auto_voice.log
```

## 3. 语音使用流程

默认唤醒词是：

```text
小智小智
```

典型流程：

```text
用户：小智小智
系统：我在
用户：把面包加入购物车
系统：好的，已加入购物车。
用户：结账
系统：好的，正在为您结账。请选择微信支付、支付宝支付或银联云闪付。
用户：微信
系统：好的，已为您打开微信收款码，请扫码支付。
```

支持的常见命令：

| 用户说法 | 行为 |
| --- | --- |
| `把面包加入购物车` | 加入面包 |
| `把牛奶加入购物车` | 加入牛奶 |
| `把水加入购物车` | 加入矿泉水 |
| `清空购物车` | 清空购物车 |
| `结账` | 进入付款方式选择 |
| `微信` / `微信支付` | 显示微信收款码 |
| `支付宝` / `支付宝支付` | 显示支付宝收款码 |

## 4. 直接测试语音命令

不通过麦克风，直接模拟一句识别结果：

```sh
sh /userdata/Embed_project/scripts/run_voiceask_speaker.sh ask '把面包加入购物车'
sh /userdata/Embed_project/scripts/run_voiceask_speaker.sh ask '结账'
sh /userdata/Embed_project/scripts/run_voiceask_speaker.sh ask '微信'
```

从电脑上通过 adb 执行：

```powershell
C:\tmp\adb.exe shell "sh /userdata/Embed_project/scripts/run_voiceask_speaker.sh ask '微信'"
```

## 5. 付款码和自动 reset

当前付款码逻辑：

1. 说 `结账` 后，系统询问付款方式。
2. 说 `微信` 显示微信收款码。
3. 说 `支付宝` 显示支付宝收款码。
4. 付款码显示 60 秒后自动关闭。
5. 自动清空购物车。
6. 自动回到主界面，等待下一位顾客。

相关临时状态文件：

```sh
/tmp/qsm_retail_voice_state
/tmp/qsm_payment_waiting_method
```

正常情况下，语音命令被 UI 消费后，`/tmp/qsm_retail_voice_state` 会被删除，避免旧命令重复执行。

## 6. 防止付款码死循环

系统已经处理“喇叭播报被麦克风重新识别”的问题。

例如系统自己播报：

```text
好的，已为您打开微信收款码，请扫码支付。
```

如果这句话又被麦克风识别到，脚本会输出：

```text
Ignoring payment reply echo.
```

并且不会再次触发 `pay:wechat`，不会重复弹出微信收款码。

## 7. 查看日志

查看 LVDS/扫码/付款码界面日志：

```sh
tail -80 /userdata/Embed_project/qr_scanner_display.log
```

查看语音监听日志：

```sh
tail -80 /tmp/qsm_auto_voice.log
```

查看当前进程：

```sh
pgrep -af qr_scanner_display
pgrep -af run_voiceask_speaker
pgrep -af start_voice_auto_listen
```

## 8. 常见排障

### 没有语音反应

重启完整系统：

```sh
sh /userdata/Embed_project/scripts/run_mission.sh
```

查看语音日志：

```sh
tail -80 /tmp/qsm_auto_voice.log
```

### 付款码一直重复弹出

先清理临时状态，再重启：

```sh
rm -f /tmp/qsm_retail_voice_state /tmp/qsm_payment_waiting_method
sh /userdata/Embed_project/scripts/run_mission.sh
```

确认脚本已经有回声过滤：

```sh
sh /userdata/Embed_project/scripts/run_voiceask_speaker.sh ask '好的，已为您打开微信收款码，请扫码支付。'
```

正常输出：

```text
Ignoring payment reply echo.
```

### LVDS 界面没有启动

查看界面日志：

```sh
tail -80 /userdata/Embed_project/qr_scanner_display.log
```

重新启动界面：

```sh
sh /userdata/Embed_project/scripts/start_retail_lvds_ui.sh
```

### 恢复桌面

```sh
sh /userdata/Embed_project/scripts/restore_lvds_desktop.sh
```

## 9. 开发侧常用命令

运行测试：

```powershell
C:\Users\cxb\.cache\codex-runtimes\codex-primary-runtime\dependencies\python\python.exe -m unittest discover -s tests
```

交叉编译 LVDS UI：

```powershell
wsl bash -lc "mkdir -p build_arm && aarch64-linux-gnu-gcc -std=c11 -O2 -Wall -D_GNU_SOURCE -I lib/quirc src/qr_scanner_display.c lib/quirc/*.c -lm -o build_arm/qr_scanner_display"
```

推送二进制到板子：

```powershell
C:\tmp\adb.exe push .\build_arm\qr_scanner_display /userdata/Embed_project/bin/qr_scanner_display
C:\tmp\adb.exe shell "chmod +x /userdata/Embed_project/bin/qr_scanner_display"
```

推送语音脚本到板子：

```powershell
C:\tmp\adb.exe push .\scripts\run_voiceask_speaker.sh /userdata/Embed_project/scripts/run_voiceask_speaker.sh
C:\tmp\adb.exe shell "chmod +x /userdata/Embed_project/scripts/run_voiceask_speaker.sh"
```
