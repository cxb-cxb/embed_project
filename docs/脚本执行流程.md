# 智能售货系统脚本执行流程

本文档按演示顺序说明当前项目脚本怎么执行。

## 1. 一键启动完整系统

在电脑 PowerShell 中执行：

```powershell
C:\tmp\adb.exe shell "sh /userdata/Embed_project/scripts/run_mission.sh"
```

该脚本会自动完成：

1. 停止旧的语音监听、播放、录音进程。
2. 写入语音参数：唤醒录音 `5s`，命令录音 `7s`。
3. 启动 LVDS 智能售货界面。
4. 启动语音唤醒监听。

正常输出会看到：

```text
[mission] run_mission is ready.
[mission] screen_log=/userdata/Embed_project/qr_scanner_display.log
[mission] voice_log=/tmp/qsm_auto_voice.log
```

## 2. 正常语音购物流程

对着板子说：

```text
小智小智
```

系统回复：

```text
我在
```

然后可以说：

```text
把面包加入购物车
把牛奶加入购物车
把水加入购物车
结账
```

说 `结账` 后，系统会询问付款方式：

```text
请选择微信支付、支付宝支付或银联云闪付
```

继续说：

```text
微信
```

或：

```text
支付宝
```

屏幕会弹出对应收款码。

## 3. 顾客付款完成后按回车

付款码弹出后，如果顾客已经扫码付款，在电脑 PowerShell 中执行：

```powershell
C:\tmp\adb.exe shell "sh /userdata/Embed_project/scripts/payment_done_enter.sh"
```

看到提示后按一次回车：

```text
Payment QR is showing: press Enter after the customer has paid.
```

按回车后，系统会立即：

1. 关闭微信/支付宝收款码。
2. 清空购物车。
3. 清空本次订单状态。
4. 回到主界面。
5. 重新等待下一位顾客。

如果不按回车，付款码会在 `60s` 后自动关闭并清空购物车。

## 4. 不通过麦克风直接测试命令

可以用 `run_voiceask_speaker.sh ask` 直接模拟语音识别结果。

```powershell
C:\tmp\adb.exe shell "sh /userdata/Embed_project/scripts/run_voiceask_speaker.sh ask '把面包加入购物车'"
C:\tmp\adb.exe shell "sh /userdata/Embed_project/scripts/run_voiceask_speaker.sh ask '结账'"
C:\tmp\adb.exe shell "sh /userdata/Embed_project/scripts/run_voiceask_speaker.sh ask '微信'"
```

模拟付款完成：

```powershell
C:\tmp\adb.exe shell "sh /userdata/Embed_project/scripts/payment_done_enter.sh"
```

然后按回车。

## 5. 常用日志

查看屏幕、扫码、购物车、付款码日志：

```powershell
C:\tmp\adb.exe shell "tail -80 /userdata/Embed_project/qr_scanner_display.log"
```

查看语音监听日志：

```powershell
C:\tmp\adb.exe shell "tail -80 /tmp/qsm_auto_voice.log"
```

查看启动日志：

```powershell
C:\tmp\adb.exe shell "cat /tmp/qsm_run_mission_last.log 2>/dev/null || true"
```

## 6. 常用状态文件

语音命令状态：

```sh
/tmp/qsm_retail_voice_state
```

等待付款方式状态：

```sh
/tmp/qsm_payment_waiting_method
```

付款完成触发文件：

```sh
/tmp/qsm_payment_done
```

正常情况下，命令被 UI 消费后会自动删除，不需要手动清理。

## 7. 出问题时重启

如果语音或界面状态异常，先清理临时状态，再重启：

```powershell
C:\tmp\adb.exe shell "rm -f /tmp/qsm_retail_voice_state /tmp/qsm_payment_waiting_method /tmp/qsm_payment_done; sh /userdata/Embed_project/scripts/run_mission.sh"
```

如果只想重启 LVDS 界面：

```powershell
C:\tmp\adb.exe shell "kill -9 $(cat /tmp/qr_scanner_display_watchdog.pid 2>/dev/null) 2>/dev/null || true; kill -9 $(cat /tmp/qr_scanner_display.pid 2>/dev/null) 2>/dev/null || true; sh /userdata/Embed_project/scripts/start_retail_lvds_ui.sh"
```

## 8. 恢复桌面

退出智能售货 LVDS 界面并恢复桌面：

```powershell
C:\tmp\adb.exe shell "sh /userdata/Embed_project/scripts/restore_lvds_desktop.sh"
```

## 9. 开发侧更新到板子

运行测试：

```powershell
C:\Users\cxb\.cache\codex-runtimes\codex-primary-runtime\dependencies\python\python.exe -m unittest discover -s tests
```

编译 LVDS UI：

```powershell
wsl bash -lc "aarch64-linux-gnu-gcc -std=c11 -O2 -Wall -D_GNU_SOURCE -I lib/quirc src/qr_scanner_display.c lib/quirc/*.c -lm -o build_arm/qr_scanner_display"
```

推送 LVDS UI 二进制：

```powershell
C:\tmp\adb.exe push .\build_arm\qr_scanner_display /userdata/Embed_project/bin/qr_scanner_display
C:\tmp\adb.exe shell "chmod +x /userdata/Embed_project/bin/qr_scanner_display"
```

推送语音脚本：

```powershell
C:\tmp\adb.exe push .\scripts\run_voiceask_speaker.sh /userdata/Embed_project/scripts/run_voiceask_speaker.sh
C:\tmp\adb.exe push .\scripts\start_voice_auto_listen.sh /userdata/Embed_project/scripts/start_voice_auto_listen.sh
C:\tmp\adb.exe push .\scripts\configure_board_voice_speaker.sh /userdata/Embed_project/scripts/configure_board_voice_speaker.sh
C:\tmp\adb.exe push .\scripts\payment_done_enter.sh /userdata/Embed_project/scripts/payment_done_enter.sh
C:\tmp\adb.exe shell "chmod +x /userdata/Embed_project/scripts/*.sh"
```
